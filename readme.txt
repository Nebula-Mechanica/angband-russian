=== Общие сведения ===

Angband - псевдографическая игра, цель которой - исследование подземелья.
Angband использует текстовые символы для представления стен подземелья и его
обитателей, аналогично таким играм как Rogue, Hack, NetHack и Moria.

Сюжет Angband основан на произведениях Дж.Р.Р.Толкиена, в особенности
"Сильмариллион". Для получения полного удовольствия от игры настоятельно
рекомендуется почитать!

В каталоге lib/help/ доступна подробная справка по игре. Ее можно вызывать
непосредственно из игры нажатием "?" (Shift+?).

Оригинальная версия Angband доступна на FTP-сайтах clockwork.dementia.org и
ftp.cis.ksu.edu. Русифицированная версия пока не имеет четкого сайта.

Сообщайте об ошибках по адресу "uav@urmail.ru". Если ошибка связана с
русификацией, она будет исправлена. Если ошибка будет присутствовать в
оригинальном Angband, сообщение о ней, возможно, будет отправлено
разработчику.

Angband работает почти на всех известных платформах. Русифицированная версия
на данный момент переведена на следующие:

	- UNIX/curses
	- UNIX/X11
	- Windows

Для прочих платформ игра теоретически может работать, но это маловероятно.
Новые порты приветствуются! В частности для DOS и Macintosh.

Скомпилировать и запустить Angband на UNIX-платформах можно так (в самом
общем случае):

	$ tar -xzvf angband-3.0.5r.tar.bz2
	$ cd angband-3.0.5r/src
	$ make -f Makefile.std
	$ cd ..
	$ src/angband -mgcu -- -b       # curses
     или
        $ src/angband -mx11             # x11

Windows-версия распространяется в готовом виде - запускайте и наслаждайтесь.

Русифицированная версия Angband может свободно читать сохраненные игры
английской версии. Обратное в целом также верно, хотя русский текст может не
сохраниться (русское имя героя, русские надписи на предметах). Сохраненные
игры русифицированной версии с другой платформы могут страдать теми же
проблемами из-за различия кодировок, но в целом также должны читаться.

Желаем удачи.
Настоятельно рекомендуется прочитать большинство файлов справки.
